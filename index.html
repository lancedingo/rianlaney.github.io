Projects

A list of previous projects built using Ruby on Rails, React.js, and WordPress.

Mediset

A group built medication reminder application, that allows the user to keep track of multiple doctors, medications, and pharmacy information. The app also sends a reminder to your phone to either take your medication or refill your prescription order. The application is built on Ruby on Rails, React.js, Bootstrap 4. Using rails as a database, and Heroku to send the reminders to your cell phone using Twilio, and also GitHub as a collaboration system.

Wyn-BA Stats

An easy to use react based NBA stat tracker, that allows the user to search either by team or by player, to see up to date stats. This was the first time using an API, application programming interface, to incorporate data that was attached to an online database and implement that data into our own website. The website was mostly front end. The application was built with React.js, JSON, HTML, and CSS.

Pawz Web Design

This was my very first website, built before I learned how to code using Wordpress backed by php. Developed for my friends whose small busisnesses are now on a world-wide market!
